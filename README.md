# CyberConnect Contracts

This hosts all contracts for CyberConnect's social graph protocol

# Dependencies

[Foundry](https://github.com/foundry-rs/foundry) ([book](https://book.getfoundry.sh/))


# Usage

To build
`forge build`

To test
`forge test`
